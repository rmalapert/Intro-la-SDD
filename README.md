# Introduction à la Science des Données - CPES Saclay

Ce dépôt contient les supports de cours et les exercices pour le cours d'Introduction à la Science des Données en première année du CPES.

## Structure du cours

Le cours est organisé par semaine :

- **Semaine 1** : Introduction, Python, Pandas, Étude statistique
- **Semaine 2** : Analyse de données (Titanic, espèces)
- **Semaine 3** : Bonnes pratiques, Jeux de données, Attributs
- **Semaine 4** : Tableaux, Images, Extraction d'attributs, Classificateurs
- **Semaine 5** : Traitement d'images
- **Semaine 6** : Diaporamas, Projet Arcimboldo
- **Semaine 7** : Analyse en Composantes Principales (ACP), Classificateurs
- **Semaines 8 et 9** : Projet final

## Exercices

Le dossier `Exercices` contient des exercices d'entraînement, notamment sur la manipulation de tableaux NumPy (`numpyarray`).

## Prérequis

- Python 3.x
- Bibliothèques : numpy, pandas, matplotlib, scikit-learn, etc.

## Utilisation

Chaque dossier de semaine contient des fichiers Markdown (`.md`) avec le contenu du cours et des exercices. Certains dossiers contiennent également des scripts Python (`utilities.py`) et des jeux de données (`.csv`).
